# 1-start.md (Начало дизайна)

Этот воркфлоу отвечает за создание первого экрана (Hero-секции) на основе референса.

## Основные инструкции

1. **Технологический стек**:
   - Использовать только чистый HTML, CSS и JavaScript.
   - Никаких фреймворков (React, Vue, Tailwind и т.д. запрещены на этом этапе).

2. **Визуальное соответствие**:
   - Создать идеальную секцию, максимально похожую на загруженную картинку-референс.
   - Соблюдать аккуратные отступы и минималистичную типографику.
   - **Микро-анимации**: Сразу продумать и реализовать легкие анимации (плавное появление текста, наведение на кнопки, легкое движение декоративных элементов) для придания "премиальности".

3. **Типографика**:
   - Найти и подобрать максимально похожие шрифты в [Google Fonts](https://fonts.google.com/):
     - Антиквы (Serif) для заголовков или акцентов.
     - Моноширинные (Monospace) для технического текста или мелких деталей.
     - Рукописные (Script) если есть в референсе.
   - **Инструкции по поиску**:
     - Используй фильтры по категориям (Serif, Sans Serif, Display, Monospace, Handwriting).
     - Сравнивай начертания (вес, высоту строчных букв, форму засечек).
     - Выбирай только нужные начертания (например, 400 и 700), чтобы не перегружать страницу.
   - **Инструкции по подключению**:
     - Подключай через `<link>` в секции `<head>` твоего HTML.
     - Указывай шрифты в CSS через переменную или напрямую в `font-family`.

4. **Иконки**:
   - Использовать иконки из открытых библиотек (например, Iconify).
   - Подключать через CDN (чтобы избежать лишних файлов svg в структуре на старте).

5. **Цель**:
   - Получить визуально "pixel-perfect" (или близкий к тому) первый экран.
