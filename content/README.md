# Контент для нового сайта Musikschule CMS Bielefeld

## Структура папок

```
content/
├── 01_start/           # Главная страница (Hero, Welcome)
├── 02_uber_uns/        # О нас - миссия, история
├── 03_angebot/         # Предложения (услуги)
│   ├── instrumente/    # Музыкальные инструменты
│   ├── gesang/         # Вокал
│   ├── kunst/          # Искусство
│   └── jekits/         # Программа JeKits
├── 04_preise/          # Цены и регистрация
├── 05_standorte/       # Местоположения
├── 06_kontakt/         # Контактная информация
└── 07_legal/           # AGB, Impressum (юридическая информация)
```

## Рекомендация для лендинга

Для лендинга можно использовать следующие секции:
1. **Hero** - Приветствие + цитата (Psalm 103)
2. **О школе** - Краткое описание миссии
3. **Услуги** - Обзор всех направлений (музыка, вокал, искусство)
4. **Программа JeKits** - Специальная программа для школ
5. **Цены** - Таблица с ценами
6. **Локации** - Карта/список адресов
7. **Контакты** - Форма связи + контакты
